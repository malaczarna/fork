1CC PS

  • git config --global user.name "" - konfiguruje globalnie nazwę użytkownika na ""
  • git add . - dodaje wszystkie zmienione pliki do commita np. zmieniłeś wiele plików i chcesz zatwierdzić wszystkie za jednym razem
  • git reset HEAD - usuwa wszystkie pliki z poczekalni zaś jeżeli np. zmieniłeś dwa pliki i chcesz zatwierdzić je osobno, ale omyłkowo dodałeś oba pliki do poczzekalni możesz usunąć wybrany plik za pomocą komendy git reset HEAD
  • git checkout "branch" - przełącza na gałąź "branch"
  • gitk --all - wyświetla program ze środowiskiem graficznym i umożliwia podgląd wszytskich zmian w repozytorium
  • git branch -merged - wypisuje wszystkie gałęzie połączone z głową gałęzi na której się znajdujemy
  • git branch "branch" "commit hash" - dodaje nową gałąź "branch" a jako HEAD ustawia "commit hash"
  • git pull "remote" "branch" - pobiera gałąź ze zdalnego repozytorium o nazwie "remote", i zapisuje ją w lokalnej gałęzi
